# sap-abap-inventory-replenishment

Smart Inventory Replenishment & Alert System with REST API Integration.

## Repository Structure

- `source/`
  - `ZREPL_CHECK.abap` – executable report entry point.
  - `ZCL_REPL_HANDLER_DEF.abap` – class definition for replenishment handling.
  - `ZCL_REPL_HANDLER.abap` – class implementation for replenishment handling.
- `table/` – ABAP table (DDIC) artifacts.
- `smartform/` – SmartForm artifacts.
- `postman/` – Postman collections and environments.
- `docs/` – project documentation.
- `test/` – test assets and related content.
- `screenshots/` – screenshots and visual references.
